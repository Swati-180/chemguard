# **App Name**: ChemGuard AI Dashboard

## Core Features:

- Secure Admin Access: User authentication and authorization to secure the admin dashboard.
- Dynamic Dashboard Overview: Display key performance indicators (KPIs) like chemical batches, shipments, IoT devices, critical alerts, and compliance scores.
- Interactive Data Visualizations: Render animated line charts for chemical usage trends, bar charts for shipment movement, and spike anomaly charts for suspicious activity detection.
- Global Shipment Tracking Map: Visualize live shipment tracking on an interactive map with animated routes and glowing nodes.
- AI-Powered Risk Analysis Tool: Utilize AI to provide an overall risk summary (Low/Medium/High) and highlight anomalies detected across systems and routes.
- Real-time Alert Management: Display a dynamic table of recent alerts with color-coded severity, timestamps, shipment IDs, types, locations, and statuses.
- Unified Navigation & Search: Provide a comprehensive left sidebar navigation and a top bar with a global search function for chemicals, shipments, and alerts.

## Style Guidelines:

- Background color: Dark, professional navy-blue (#1A2633) to set a sophisticated and futuristic tone, mirroring security operations centers.
- Primary accent color: Vibrant cyan (#2EDEFF) for key data, interactive elements, and general highlighting, providing a distinct digital glow against the dark background.
- Secondary accent color: Neon green-cyan (#32FAC0) used for status indicators (e.g., 'excellent standing', 'online') and positive trend visual cues. Additional neon colors like red (#FF6B6B), orange (#FFA066), and purple (#A08CFF) will be used specifically for alerts and severity indicators (e.g., critical, warning, info).
- Headline font: 'Space Grotesk' (sans-serif) for its modern, tech-inspired, and slightly scientific feel. Body text font: 'Inter' (sans-serif) for clean readability and objective clarity across all data displays.
- Use sharp, outline-style vector icons that align with a minimalist and technical aesthetic. Icons for KPIs, navigation, and device statuses should be consistent in style.
- Adopt a 'glassmorphism' design for cards and panels, creating a translucent, layered effect with frosted backgrounds. All UI elements, including panels and buttons, should feature smooth rounded corners for a modern, approachable feel within the professional dark theme.
- Implement subtle, smooth transitions and micro-animations for data updates in charts, navigation interactions, and glowing effects for nodes/routes on the global tracking map, enhancing the futuristic and live monitoring experience.
